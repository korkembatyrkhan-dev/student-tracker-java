# Student Tracker Java1

## Description

Student Tracker is a console application written in Java Core for managing students, courses and grades.

The application allows users to:

- Add students
- Add courses
- Assign grades
- Calculate student GPA
- Show Top 3 students
- Show course analytics
- Display students above a GPA threshold

## Technologies

- Java
- OOP
- ArrayList
- HashMap
- Scanner

## How to Run

1. Open the project in IntelliJ IDEA.
2. Open Main.java.
3. Run the main() method.
4. Use the console menu.
